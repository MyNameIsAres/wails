# 日志

## 概览

Wails 运行时提供了一个可以从 Go 或 Javascript 调用的日志机制。 像大多数 日志记录器一样，有一些日志级别：

  - 轨迹
  - Debug
  - 信息
  - 警告
  - 错误
  - Fatal

日志记录器将在当前或更高的日志级别输出任何日志消息。 例如： `调试` 日志 级别将输出除 `追踪` 消息以外的所有消息。

### 日志打印

去签署: `LogPrint(ctx context.Context, message string)`

JS 签署： `LogPrint(消息：字符串)`

将给定的消息记录为原始消息。

### LogPrintf

去签署: `LogPrintf(ctx context. Context, format string, args ...interface{})`

将给定的消息记录为原始消息。

### LogTrace

去签署: `LogTrace(ctx context.Context, message string)`

JS 签署： `LogTrace(消息：字符串)`

在 `跟踪` 日志级别记录给定的消息。

### LogTracef

去签署: `LogTracef(ctx context. Context, format string, args ...interface{})`

在 `跟踪` 日志级别记录给定的消息。

### LogDebug

去签署: `LogDebug(ctx context.Context, message 字符串)`

JS 签署： `LogDebug(消息：字符串)`

在 `调试` 日志级别记录给定的消息。

### LogDebugf

去签署: `LogDebugf(ctx context. Context, format string, args ...interface{})`

在 `调试` 日志级别记录给定的消息。

### LogInfo

去签署: `LogInfo(ctx context.Context, message 字符串)`

JS 签署： `LogInfo(消息：字符串)`

在 `信息` 日志级别记录给定的消息。

### LogInfof

去签署: `LogInfof(ctx context. Context, format string, args ...interface{})`

在 `信息` 日志级别记录给定的消息。

### 日志警告

去签署: `LogWarning(ctx context.Context, message string)`

JS 签署： `Log警告(消息：字符串)`

在 `警告` 日志级别记录给定的消息。

### 日志警告

去签署: `LogWarningf(ctx context. Context, format string, args ...interface{})`

在 `警告` 日志级别记录给定的消息。

### LogError

去签署: `LogError(ctx context.Context, message string)`

JS 签署： `LogError(消息：字符串)`

日志记录给定的消息到 `错误` 日志。

### LogErrorf

去签署: `LogErrorf(ctx context. Context, format string, args ...interface{})`

日志记录给定的消息到 `错误` 日志。

### 致命的

去签署: `LogFatal(ctx context.Context, message 字符串)`

JS 签署： `LogFatal(消息：字符串)`

将给定消息记录到 `致命` 日志。

### LogFatalf

去签署: `LogFatalf(ctx context. Context, format string, args ...interface{})`

将给定消息记录到 `致命` 日志。

### 日志设置日志日志级别

去签署: `LogSetLevel(ctx context.Context, level logger.LogLevel)`

JS 签署： `LogSetLevel(级别：数字)`

设置日志级别。 在 Javascript 中，数字与以下日志级别有关：

| 值 | 日志级别  |
| - | ----- |
| 1 | 轨迹    |
| 2 | Debug |
| 3 | 信息    |
| 4 | 警告    |
| 5 | 错误    |

## 使用自定义日志记录

通过使用 [日志](../options.mdx#logger) 应用程序选项提供一个自定义日志。 唯一的要求是记录器实现 `logger.Logger` 接口 定义在 `github.com/wailsapp/wails/v2/pkg/logger`

```go title="logger.go"
输入Logger 接口 然后
    Print(消息字符串)
    Trace(消息字符串)
    Debug(消息字符串)
    Info(消息字符串)
    警告 (消息字符串)
    错误(消息字符串)
    Fatal(消息字符串)
}
```
