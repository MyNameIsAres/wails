# 一. 导言

运行时间是一个为您的应用程序提供实用方法的库。 有一个 Go 和 Javascript 运行时间 ，其目标是尽可能地尝试保持对等。

Go Runtime 可以通过导入 `github.com/wailsapp/wails/v2/pkg/runtime` 来使用。 这个包 中的所有方法都以上下文作为第一个参数。 此上下文应该从 [On启动](../options.mdx#onstartup) 或 [OnDomReady](../options.mdx#ondomready) 钩子。

:::info 笔记

虽然上下文将提供给 [Onstartup](../options.mdx#onstartup) 方法， 无法保证运行时间将在这个方法中工作，因为 窗口正在另一个线程初始化。 如果 您想在启动时调用运行方式, 请使用 [OnDomReady](../options.mdx#ondomready)。

:::

Javascript 库可以通过 `window 运行时` 地图到前端。 当使用 `dev` 模式提供运行时生成了一个运行时的软件包。 这应该位于您 前端目录中的 `wailsjs` 目录。

### 隐藏

去签署: `隐藏 (ctx context.Context)`

隐藏应用程序。

:::info Note On Mac, this will hide the application in the same way as the `Hide` menu item in standard Mac applications. 隐藏窗口不同，但应用程序仍处于前台状态。 对于Windows和Linux，这目前与 `Windows隐藏` 相同。 :::

### 显示

去签署: `show(ctx context.Context)`

显示应用程序。

::info 笔记 在 Mac 上，这将使应用程序返回前台。 对于Windows和Linux，这目前与 `Windowshop` 相同。 :::

### 退出

去签署: `Quit(ctx context.Context)`

退出应用程序。

### 环境

去签署: `Environment(ctx context.Context) EnvironmentInfo`

返回当前环境的详细信息。
