# 菜单

## 概览

这些方法与应用程序菜单有关。

:::info Javascript
  当前在 JS 运行时不支持菜单。
:::

### 菜单设置应用程序菜单
去签署: `MenuSetApplicationMenu(ctx context. Context, menu *menu.Menu)`

设置应用程序菜单到给定的 [菜单](../menus.mdx)。

### 菜单更新应用程序菜单
去签署: `MenuUpdateApplicationMenu(ctx context.Context)`

更新应用程序菜单，选取对 `菜单设置应用程序菜单` 的任何更改。
