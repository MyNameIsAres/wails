# 窗口

## 概览

这些方法能够控制应用程序窗口。

### 窗口设置标题
去签署: `WindowSetTitle(ctx context. Context, title string)`

JS 签署： `WindowSetTitle(标题：字符串)`

设置窗口标题栏中的文本。

### Windows全屏
去签署: `WindowFullscreen(ctx context.Context)`

JS 签署： `WindowFullscreen()`

使窗口全屏化。

### Windows全屏
去签署: `WindowUnfullscreen(ctx context.Context)`

JS 签署： `WindowUnfullscreen()`

在全屏之前还原上一个窗口尺寸和位置。

### 窗口中心
去签署: `WindowCenter(ctx context.Context)`

JS 签署： `WindowCenter()`

将窗口放在监视器上的窗口上。

### 窗口重新加载
去签署: `WindowReload(ctx context.Context)`

JS 签署： `WindowReload()`

执行"重新加载"(重新加载当前页面)。

### WindowReloadApp
去签署: `WindowReloadApp(ctx context.Context)`

JS 签署： `WindowReloadApp()`

重新加载应用程序前端。

### WindowsSetSy

### WindowsSet系统默认主题
去签署: `WindowSetSystemDefaultTheme(ctx context)`

JS 签署： `WindowSetSystemDefaultTheme()`

仅限窗口

设置窗口主题为系统默认 (dark/light).

### WindowsSetLight主题
去签署: `WindowSetLightTheme(ctx context.Context)`

JS 签署： `WindowSetLightTheme()`

仅限窗口

将窗口主题设置为亮点。

### WindowsSet暗色主题
去签署: `WindowSetDarkTheme(ctx context.Context)`

JS 签署： `WindowSetDarkTheme()`

仅限窗口

将窗口主题设置为暗淡。

### 窗口显示
去签署: `WindowShow(ctx context.Context)`

JS 签署： `WindowShow()`

显示当前隐藏的窗口。

### 窗口隐藏
去签署: `WindowHide(ctx context.Context)`

JS 签署： `WindowHide()`

隐藏窗口，如果它目前可见。

### 窗口设置大小
去签署: `WindowSetSize(ctx context. Context, 宽度, 高度)`

JS 签署： `WindowSetSize(大小：大小)`

设置窗口的宽度和高度

### 窗口Getsize
去签署: `WindowGetSize(ctx context) (宽度, 高度)`

JS 签署： `WindowGetSize() : 大小`

获取窗口的宽度和高度

### Windows设置Minize
去签署: `WindowSetMinSize(ctx context, 宽度, 高度)`

JS 签署： `WindowSetMinize(大小：大小)`

设置最小窗口大小。 如果窗口当前小于给定的尺寸，将调整窗口的大小。

设置大小为 `0,0` 将禁用此限制。

### Windows设置MaxSize
去签署: `WindowSetMaxSize(ctx context. Context, 宽度, 高度)`

JS 签署： `WindowSetMaxSize(大小：大小)`

设置最大窗口大小。 如果窗口当前大于给定的尺寸，将调整窗口的大小。

设置大小为 `0,0` 将禁用此限制。

### Windows SetAlwaysOntop
去签署: `WindowSetAlwaysOnTop(ctx context.Context, b bool)`

JS 签署： `WindowSetAlwaysOnTop(b: Boolen)`

设置窗口总是在顶部或不在顶部。


### 窗口设置位置
签署: `Windows设置位置(ctx context.Context, x int, y int)`

JS 签署： `Windows定位(位置：位置)`

设置窗口相对于当前显示器的位置。

### 窗口获取位置
去签署: `WindowGetPosition(ctx context.Context) (x int, y int)`

JS 签署： `WindowGetPosition() : 位置`

获取窗口相对于当前显示器的位置。

### Windows最大化
去签署: `WindowMaximise(ctx context.Context)`

JS 签署： `WindowMaximise()`

最大化窗口以填充屏幕。

### 窗口最大化
去签署: `WindowUnmaximise(ctx context.Context)`

JS 签署： `WindowUnmaximise()`

在最大化之前将窗口还原到尺寸和位置。

### Windows切换最大化
去签署: `WindowToggleMaximise(ctx context.Context)`

JS 签署： `WindowToggleMaximise()`

在最大化和未最大化之间切换。

### 窗口最小化
去签署: `WindowsMinimise(ctx context.Context)`

JS 签署： `Windows Minimise()`

最小化窗口

### 窗口最小化
去签署: `WindowUnminimise(ctx context.Context)`

JS 签署： `WindowUnminimise()`

将窗口还原到最小化前的尺寸和位置。

### WindowSetBackgroundColour
去签署: `Windows SetBackgroundColourour(ctx context.Context, R, G, B, A uint8)`

JS 签署： `WindowSetBackgroundColour(R, G, B, A)`

将窗口的背景颜色设置为给定的 RGBA 颜色定义。 此颜色将会显示所有透明像素。

R、G、B和A有效值为0-255。

:::info 窗口

在 Windows上，仅支持 0 或 255 的alpha 值。 不为 0 的任何值将被视为255。

:::
