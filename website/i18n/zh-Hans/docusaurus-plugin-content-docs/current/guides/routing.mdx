
# 路由

路由是在应用程序中切换视图的流行方式。 本页提供了一些关于如何做到这一点的指导。

## Vue

Vue 中推荐的路由方式是 [散列模式](https://next.router.vuejs.org/guide/essentials/history-mode.html#hash-mode):

```js
import { createRouter, createWebHashHistory } from 'vue-router'

const router = createRouter({
  history: createWebHashHistory(),
  routes: [
    //...
  ],
})
```

## 角形

建议的角度路由方式是 [哈希位置策略](https://codecraft.tv/courses/angular/routing/routing-strategies#_hashlocationstrategy):

```ts
RouterModule.forRoot(路由， {useHash: true})
```
