# 鼠标按钮

Wails运行时拦截鼠标点击，以确定一个无帧的窗口是否需要调整大小或需要移动一个窗口。 有人问它如何检测鼠标点击何时发生，因为 `wind.onclick` 没有正确地报告鼠标按钮。 下面的代码显示如何检测鼠标点击：

```javascript

window.addEventListener('mousedown', handleMouseButtonDown);

function handleMouseButtonDown(event) {
  if (event.button === 0) {
    // left mouse button
  } else if (event.button === 1) {
    // middle mouse button
  } else if (event.button === 2) {
    // right mouse button
  } else if (event.button === 3) {
    // back mouse button
  } else if (event.button === 4) {
    // forward mouse button
  } else {
    // other mouse button
  }
}

```
参考： https://developer.mozilla.org/en-US/docs/Web/ADI/MouseEvent/button
