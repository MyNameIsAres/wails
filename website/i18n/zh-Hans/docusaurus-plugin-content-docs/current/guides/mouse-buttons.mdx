# 鼠标按钮

The Wails runtime intercepts mouse clicks to determine whether a frameless window needs resizing or a window needs to be moved. It has been asked how to detect when a mouse click has occurred, because `window.onclick` doesn't report the mouse buttons correctly. The following code shows how to detect mouse clicks: 有人问它如何检测鼠标点击何时发生，因为 `wind.onclick` 没有正确地报告鼠标按钮。 下面的代码显示如何检测鼠标点击：

```javascript

window.addEventListener('mousedown', handleMouseButtonDown);

function handleMouseButtonDown(event) {
  if (event.button === 0) {
    // left mouse button
  } else if (event.button === 1) {
    // middle mouse button
  } else if (event.button === 2) {
    // right mouse button
  } else if (event.button === 3) {
    // back mouse button
  } else if (event.button === 4) {
    // forward mouse button
  } else {
    // other mouse button
  }
}

```
Reference: https://developer.mozilla.org/en-US/docs/Web/API/MouseEvent/button
