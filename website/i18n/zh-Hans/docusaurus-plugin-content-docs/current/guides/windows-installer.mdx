Wails支持使用 [NSIS 安装程序](https://nsis.sourceforge.io/) 生成Windows安装程序。

## Installing NSIS

### 窗口

安装程序可在 [NSS Download](https://nsis.sourceforge.io/Download) 页面上使用。

如果您使用巧克力包管理器，请运行下面的脚本：
```
choco install nsis
```
如果您手动安装 NSIS，您需要添加 *Bin* 文件夹，其中包含 `makensis。 xe`, 在您的 NSIS 安装到您的路径中。 [在这里](https://www.architectryan.com/2018/03/17/add-to-the-path-on-windows-10/) 是一个关于如何添加到Windows上的路径的良好教程。

### Linux

`nsi` 软件包应该可以通过您的发行版软件包管理器获得。

### MacOS

NSIS 可用来通过 homebrew 安装: `brew install nsis`

## 生成安装程序

当创建一个新项目时，Wail将生成 `build/windows/installer` 中的 NSIS 配置文件。 配置 数据从 `installer/info.json` 读取，这个配置被配置为使用项目的 `wails.json` 信息部分：

```json
// ...
 "Info":
    "companyName": "我的公司名称",
    "productName": "Wails Vite",
    "productVersion": "1.0.0",
    "copyright": "Copyright......",
    "comments": "使用Wails (https://wails.io)",
},
```

若要为您的应用程序生成安装程序，请使用 `-nsis` 旗帜与 `wails Building`：
```
线条生成-nsis
```

现在安装程序将在 `build/bin` 目录中可用。
