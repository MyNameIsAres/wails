# B. 派遣方式

Wails 是一个开放源码、社区驱动的项目。 我们欢迎任何人加入我们 ，为该项目做出贡献。 This documentation is aimed at anyone wishing to get familiar with the project and the development processes.

为该项目做出贡献的方式有多种：

- 开发新功能
- 修复bugs
- 测试
- 文档功能
- 写入教程/指南
- 帮助其他人处理问题 + 讨论板

这些指南已在各自的章节中创建。 在开始前， 请在 [贡献给邮件](https://github.com/wailsapp/wails/discussions/1520) 讨论中介绍你自己。
