# 测试

测试对确保项目质量至关重要。 测试可以真正帮助项目的一些 场景：

- 测试是否在您的本地系统上可重置错误
- 测试PR以确保其正常工作

如果您选择测试某人的错误报告是否可在您的本地系统中复制。 然后 可以随时在工单上添加一条评论, 确认这个, 并输入 `wails doctor` 的输出。

若要测试 PR，请选择 PR 来测试和检查 PR 描述是否包含 的测试场景。 如果没有，请要求打开菲律宾共和国的人提供该名单。 Once you have determined a valid test scenario, please report your findings on the PR.

如果您需要更多的清晰度或测试帮助， 请在 [贡献给Wails](https://github.com/wailsapp/wails/discussions/1520) 讨论中或在slack中提问。
