# 开发您的应用程序

您可以通过从您的项目目录运行 `wails dev` 来运行您的应用程序。 这将做以下事情：

  - 构建您的应用程序并运行它
  - 绑定您的转码到前端，以便可以从 Javascript 调用
  - 使用 [vite](https://vitejs.dev/)的力量, 将监视你的 Go 文件中的修改, 并在更改时重建/重新运行
  - 设置一个 [webserver](http://localhost:34115) 将通过浏览器为您的应用程序服务。 这允许您使用您最喜欢的浏览器扩展。 您甚至可以从控制台调用您的 Go 代码

要开始，请在项目目录中运行 `wails dev`。 关于这个问题的更多信息可以在这里找到 [](../reference/cli.mdx#dev)。
