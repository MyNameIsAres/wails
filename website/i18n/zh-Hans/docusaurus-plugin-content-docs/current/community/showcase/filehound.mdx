[FileHound 导出工具](https://www.filehound.co.uk/) FileHound 是一个云端文档管理平台，用于安全保存文件、业务流程自动化和智能捕获功能。

FileHound导出工具允许FileHound管理员能够为备份和恢复的目的运行安全的文档和数据提取任务。 此应用程序将根据您选择的筛选器下载保存在FileHound中的所有文档和/或元数据。 元数据将以 JSON 和 XML 格式导出。

后端构建于： 去1.15 Wails 1.11.0 go-sqlite3 1.14.6 go-linq 3.2

前端： Vue 2.6.11 Vuex 3.4.0 类型脚本 Tailwind1.9.6
