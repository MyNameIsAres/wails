**[cryptEasy](https://www.encrypteasy.app) 是一个简单易用的 PGP 加密工具，管理您和您的联系人的所有密钥。 加密应该是简单的。 开发于Wails。**

使用 PGP 加密消息是行业标准。 每个人都有私人和公共钥匙。 您的私钥很好，需要保持私密性，只有您才能阅读消息。 您的公钥已经分发给任何想要向您发送秘密、加密消息的人。 管理密钥、加密信件和解密信息应该是一个顺利的体验。 加密简单就是让它变得容易。
