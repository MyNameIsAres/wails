# 模板

此页面是社区支持模板的列表。 请提交一个 PR (点击 `在底部编辑此页面` ) 以包含您的模板。 要构建您自己的模板，请查看 [模板](../guides/templates.mdx) 指南。

若要使用这些模板，运行 `wils init -n "你的项目名称" -t [链接[@version]]`

如果没有版本后缀，默认使用主分支代码模板。 如果有版本后缀，则使用与此版本标签相对应的代码模板。

示例： `wails init -n "您的项目名称" -t https://github.com/misitebao/wails-template-vue`

:::警告注意事项

**Wails项目不维护，对第三方模板不负责任或责任！**

如果您对模板不确定，请检查 `package.json` 和 `wails.json` 查看哪些脚本正在运行和哪些软件包已安装。

:::

## Vue

- [wails-template-vue](https://github.com/misitebao/wails-template-vue) - 使用 Vite,Vue 和 Vue-Router(支持JavaScript 和 TypeScript) 的模板
- [wails-vite-vue-ts](https://github.com/codydbentley/wails-vite-vue-ts) - 带有Vite 的3类型脚本(和添加功能的说明)
- [ways-vute-vue-the-work](https://github.com/codydbentley/wails-vite-vue-the-works) - Vue 3 TypeScript with Vite、Vuex、Vue Router、Sass 和 ESLint + Prettier

## 角形

- [wails-angular-template](https://github.com/TAINCER/wails-angular-template) - 与TypeScript, Sass, Hot-Reload, Code-Splitting 和 i18n 的角度

## React

- [wails-react-template](https://github.com/AlienRecall/wails-react-template) - 一个使用 Reactjs 的模板
- [wails-react-template](https://github.com/flin7/wails-react-template) - 支持在线开发的React最小模板
- [wails-template-nextjs](https://github.com/LGiki/wails-template-nextjs) - 使用 Next.js 和 TypeScript 的模板

## Svelte

- [wails-svelte模板](https://github.com/raitonoberu/wails-svelte-template) - 使用 Svelte 的模板
- [wails-vite-svelte模板](https://github.com/BillBuilt/wails-vite-svelte-template) - 使用 Svelte 和 Vite 的模板
- [wails-vite-svelte-尾风模板](https://github.com/BillBuilt/wails-vite-svelte-tailwind-template) - 使用 Svelte 和 Vite 的 TailwindCSS v3 的模板

## Elm

- [was-elm-template](https://github.com/benjamin-thomas/wails-elm-template) - 开发您的 GUI 应用，配备功能编程和 **snappy** 热重新加载设置 :tada: :rocket:
